UI Design Rules

1. Tema:
   Edukatif, sejarah Indonesia, modern dengan sentuhan vintage.

2. Warna:
   - Background: krem
   - Primary: navy
   - Accent: emas
   - Success: hijau
   - Danger: merah

3. Font:
   - Judul: serif
   - Isi: sans-serif

4. Board:
   - Selalu 10×10
   - Rasio 1:1
   - Frame kayu
   - Background peta Indonesia transparan

5. Tile:
   - Nomor di kiri atas
   - Ikon di tengah
   - Tidak menggunakan teks besar seperti "Snake" atau "Quiz"

6. Sidebar:
   - Lebar tetap
   - Card konsisten
   - Tombol utama berwarna primary

7. Constraint:
   - Jangan mengubah Business Logic.
   - Jangan mengubah TypeScript types.
   - Fokus hanya pada Presentation Layer.
